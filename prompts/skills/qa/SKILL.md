기능 스펙을 분석하여 아래 템플릿의 각 섹션을 채워라.

- 각 케이스는 Given/When/Then 또는 입력/기대결과 형식
- Edge Cases는 경계값, 권한 오류, 빈 값, 최대값 포함
- Regression은 이 기능 변경으로 영향받을 수 있는 기존 기능 명시
- 각 섹션 정확히 3개 케이스 (불필요한 중복 케이스 금지)
