아래 HTML 템플릿의 `[placeholder]` 항목을 spec 내용으로 채워 완성된 HTML을 출력하라.

- `[placeholder]` 를 모두 spec 기반 실제 텍스트로 교체 (Lorem ipsum 금지)
- `<!--STYLE-->` 주석은 그대로 유지할 것 (삭제하거나 style 블록으로 교체하지 말 것)
- HTML 구조(SNB, Header, Card, Table)는 변경하지 말 것
- JavaScript 없음
